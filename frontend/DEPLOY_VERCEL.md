# نشر ربوع على Vercel

## خطوات سريعة

1. **ادخلي إلى [vercel.com](https://vercel.com)** وسجّلي الدخول (أو أنشئي حساباً بـ "Continue with GitHub").

2. **اضغطي "Add New..." → "Project"**.

3. **اختيار المستودع:**  
   اختاري `SarahO10/gp-robou` من القائمة (إن لم يظهر، اضغطي "Import Git Repository" وربطي حساب GitHub).

4. **إعدادات المشروع (مهم):**
   - **Root Directory:** اضغطي "Edit" بجانبه واختي **`frontend`** ثم "Continue".
   - باقي الخيارات اتركيها كما هي (Framework Preset: Next.js يظهر تلقائياً).

5. **Environment Variables (اختياري):**  
   إذا حابّة الخريطة أو صفقات حقيقية تظهر، أضيفي في Vercel:
   - `NEXT_PUBLIC_GOOGLE_MAPS_API_KEY` = مفتاح خرائط جوجل (للخريطة)
   - يمكن ترك الحقول الأخرى فارغة للتجربة.

6. **اضغطي "Deploy"** وانتظري حتى ينتهي البناء (دقيقتين تقريباً).

7. **الرابط:** بعد النجاح راح يطلع لك رابط مثل:  
   `https://gp-robou-xxx.vercel.app`  
   — أرسليه لصديقاتك.

---

**ملاحظة:** بدون مفاتيح API، الموقع يشتغل كامل لكن الخريطة قد لا تظهر وصفقات الأراضي تعرض بيانات تجريبية. الباك اند (التقدير والتحليلات الكاملة) يحتاج سيرفر منفصل.
